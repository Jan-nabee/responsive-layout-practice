# Responsive Layout Practice

반응형 웹 레이아웃 연습 프로젝트입니다.  
미디어 쿼리를 사용해 700px 이하 화면에서 레이아웃이 변합니다.

## 💡 주요 기능
- Flexbox 기반 레이아웃
- 반응형 메뉴 구조
- aside 영역이 모바일에서 콘텐츠 하단으로 이동

## 🖥️ 화면 예시
- 데스크탑: 메뉴 가로 정렬
- 모바일: 메뉴 2열 정렬 + 콘텐츠 단일 컬럼
